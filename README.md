# Web Scraping - Wikipedia Companies

## Project Overview
Scraped Wikipedia's list of largest companies in the United States by revenue using Python and BeautifulSoup.

## Skills Used
- Python (BeautifulSoup, Requests)
- Web Scraping
- Pandas (Data Cleaning)
- CSV Export

## Files
- `Scraping a Table From Website.ipynb` — Jupyter notebook with scraping code
- `Companies.csv` — Scraped data

## What I Learned
- Sending HTTP requests with headers (User-Agent)
- Parsing HTML with BeautifulSoup
- Extracting tables from Wikipedia
- Exporting data to CSV
